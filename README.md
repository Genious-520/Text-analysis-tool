# Text Analysis Tool (Java)

This is a Java-based text analysis application that processes user input and provides useful insights.

## Features
- Character count
- Word count
- Most common character
- Character frequency (case-insensitive)
- Word frequency (case-insensitive)
- Unique words count

## Technologies Used
- Java
- HashMap
- HashSet

## How to Run
1. Compile the program
2. Run the main class
3. Enter text input in the console
